GSD Orange

Primary Orange

Logo

Logo Set


We love our logo, so please use it responsibly. Here are some basic rules to follow:


  • Only use the approved logo assets
  • Maintain clear space around the logo (at least 20px), it needs some breathing room from other elements
  • Maintain contrast with background, avoid using over busy backgrounds

Primary Logo

The primary ClockShark logo to be used across all of ClockShark branding and communications. Available in blue and grey, and white

Clear Space

To maintain legibility of the logo always allow for the minimum clear space around the logo and between the logo elements

Wordmark

When the full logo is too small to be recognizable use the wordmark to make sure the ClockShark name is legible.

Minimum Size

The minimum width for the primary logo is 45px, anything smaller and the detail in the ace icon loses legibilty. If you need a smaller logo use the ClockShark wordmark.
